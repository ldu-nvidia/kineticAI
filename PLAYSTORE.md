## App Name

## Short Description (80 chars max)
AI-powered real-time sports coaching with IoT motion sensors.

## Full Description (4000 chars max)

KineticAI turns your phone and wearable sensors into an AI-powered coaching system that analyzes your movements in real-time and helps you improve faster.

🎿 SKI & SNOWBOARD COACHING
• Real-time technique analysis with 34 derived metrics
• Kinetic Score (0–200) composite performance rating
• Turn-by-turn coaching with phase-specific feedback
• 22 ski drills + 24 snowboard drills from PSIA/AASI methodology
• Beginner to expert progression system
• Left/right turn asymmetry analysis
• Fatigue detection across your session

📡 MULTI-SENSOR FUSION
• Dual-boot Bluetooth IMU sensors for left/right comparison
• 3-segment kinematic chain: ski → ankle → knee biomechanics
• Microphone-based snow surface classification (ice, powder, groomed)
• Thermal camera rear proximity detection
• GPS trajectory with full run reconstruction
• 200Hz high-resolution local recording

🧠 AI-POWERED INSIGHTS
• 5-layer insight engine: Orientation → Events → Metrics → Intelligence → Coaching
• Snow-adaptive coaching (different tips for ice vs powder)
• Post-run AI analysis powered by cloud LLM
• Natural language Q&A about your performance
• Adaptive scoring based on your personal baseline

⛷️ LIVE SESSION FEATURES
• Real-time weather, UV index, and frostbite risk
• Avalanche danger ratings and safety alerts
• Nearest resort info (lifts, trails, snow depth)
• Shareable run cards for social media
• Full GPS trajectory maps

🔧 HARDWARE SUPPORT
• Works with phone sensors only (no external hardware required)
• Optional: M5StickC Plus2 boot sensors for advanced metrics
• BLE 4.2 connection to up to 2 external sensors simultaneously
• WiFi bulk data transfer for post-session deep analysis

🔒 PRIVACY FIRST
• All sensor processing happens on your device
• No account required
• No personal data collected
• Optional cloud features are opt-in only

KineticAI is open-source and built for athletes who want data-driven improvement.
